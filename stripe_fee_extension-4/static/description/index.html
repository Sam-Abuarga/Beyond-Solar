Stripe Fees Extension

With Stripe Fees Extension application of Craftsync Technologies you can collect stripe processing fees from your customers.

  • Community
  • Enterprise
  • Odoo.sh

Features

  • Customer can pay Stripe transaction processing FEES while shopping.
  • Customer can pay Stripe transaction processing FEES while paying for Invoice.

Stripe Acquirer Configuration

  • Configure Stripe Payment Acquirer: Invoicing > Configuration > Payments > Payment Acquirer > Stripe

  • Enter valid Secret Key and Publishable Key under Credentials tab.

  • Check the checkbox "Add Extra Fees" under Fees tab to collect fees from customer.

  • Enter fees in Fixed and Percentage for Domestic and International


    • Stripe Fees while Shopping:

  • Stripe fees automatically will be calculated based on fixed and percentage value configured on payment acquirer.

  • Here, Fees $ 1.47 will be charged for $ 39.00 order value.

  • Here customer needs to enter credit card details to pay for Order value + Sripte Fees.

  • $ 39.00 + $ 1.47 = $ 40.47


  • Order is confirmed


    • Transaction in backend

  • Admin can see the transaction in backend. Transaction shows all values regarding order, fees, acquirer and acquirer reference.

    • Transaction in Stripe Account

    Stripe Fees while Paying for Invoice

  • Customer can pay invoice from My Account > Invoices

  • INV/2019/0005 invoice amount is $ 115.00 and stripe fees is $ 3.74.

  • Enter Credit card details and click on Pay Now button.


  • Invoice INV/2019/0005 is fully PAID.


    • Transaction in backend

  • Admin can see the transaction in backend. Transaction shows all values regarding Invoice, fees, acquirer and acquirer reference.

    • Transaction in Stripe Account
