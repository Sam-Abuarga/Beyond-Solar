Global Search

This application lets you search any record that you have configured relevant models for.
For example, a sales person can search their customers and sales orders, or a project manager can search for projects & relevant tasks. This feature needs to be enabled & configured by the administrator.

This module is also compatible with our odoo v13 Allure Theme

This module is compatible with

  • Community
  • Enterprise
  • Odoo.sh

Key Features

    Template creation for easy use.
    Batch configuration option for multiple users.
    Users can search fields by N-Levels like Many-2-One, One-2-Many & Many-2-Many.
    Search results will be redirected to a detailed form view.

Configuration

Menu Sequence: Global Search > Configuration > Settings
Users can create different kinds of templates for repetitive fields.

Global Search Batches

Menu Sequence: Global Search > Global Search Batches
Users can create & bifurcated search batches user wise.

User assignation

After creating templates users can assign those template with multiple users.
Users will be able to see only assigned templates.

Global Search

After assignation different record will be created user wise.

How does it work?

Users can search any record that they have configured relevant models for.
Click on the search icon and type it into a search box that appears.Users can see match results in the drop-down list.
